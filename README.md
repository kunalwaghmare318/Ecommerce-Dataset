# Ecommerce Database Management System

## Student Information

**Name:** Kunal Waghmare

## Project Overview

This project demonstrates the design and implementation of an Ecommerce Database Management System using MySQL. The database is designed to manage customers, products, orders, shipments, coupons, reviews, and shopping cart functionality.

## Database Tables

The project includes the following tables:

* Users
* Customers
* Products
* Orders
* OrderDetails
* Coupons
* OrderCoupons
* ProductReviews
* Shipments
* Cart

## Features

* Customer Management
* Product Management
* Order Processing
* Shipment Tracking
* Product Reviews and Ratings
* Coupon and Discount Management
* Shopping Cart Management

## SQL Queries Implemented

### Query 1: Average Product Price by Category

Displays the average price of products for each category and shows only categories whose average price is greater than 100.

### Query 7: Order Details View

Creates a view that combines customer information and order details using SQL JOIN operations.

### Query 13: Delivery Time Analysis

Calculates the delivery time for completed orders using the DATEDIFF() function.
